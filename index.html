Crafting Interactive Web Experiences.

This website is a creative showcase of animations built using GSAP (GreenSock Animation Platform), a powerful JavaScript library for high-performance animations. It brings scroll-triggered motion, smooth transitions, and dynamic effects to life. The goal is to demonstrate how modern web experiences can be enhanced with interactive and responsive visuals — all powered by clean code and scroll-based animation magic.

About This project

This website is designed to explore and showcase the power of scroll-based animations using GSAP. Every element on the page responds to user interaction and scroll behavior, creating a smooth and immersive experience. From motion effects to dynamic transformations, this project demonstrates how animation can enhance storytelling and engagement on the web.
